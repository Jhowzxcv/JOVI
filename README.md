# Foco — A câmera que estuda com você

Projeto desenvolvido para o **JOVI Hackathon 2026** pela equipe **LJP Solutions**.

## Sobre o projeto

Foco transforma a câmera do smartphone em um assistente de estudos inteligente. Ao fotografar slides, quadros ou anotações, a IA gera automaticamente resumos, flashcards e exercícios adaptativos, organizados por matéria.

## Funcionalidades

- Captura inteligente com reconhecimento de conteúdo acadêmico
- Geração automática de resumos via IA (Gemini 1.5 Flash)
- Flashcards com repetição espaçada (algoritmo FSRS)
- Quiz adaptativo baseado em IRT (Item Response Theory)
- Relatórios de desempenho por matéria
- Modo Aula para captura contínua durante as aulas
- Integração com a plataforma JOVI (vivo Mobile)

## Tecnologias utilizadas

- HTML5 semântico
- CSS3 com Flexbox
- JavaScript puro (Vanilla JS — sem frameworks ou bibliotecas externas)
